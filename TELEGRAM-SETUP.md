# Настройка Telegram-бота для приёма заявок

Заявки с формы будут приходить в Telegram-чат через специально созданного бота. Создание занимает ~5 минут.

---

## Шаг 1. Создать бота через @BotFather

1. Открой Telegram, в поиске вверху набери **`@BotFather`** и нажми на верхний результат с галочкой ✓.
2. Нажми **«Запустить»** (или отправь команду `/start`).
3. Отправь команду **`/newbot`**.
4. BotFather спросит **имя бота** (что увидит человек). Напиши, например:
   ```
   Добрый Питер — заявки
   ```
5. Затем спросит **юзернейм** (адрес бота, должен заканчиваться на `bot`). Например:
   ```
   dobrypiter_zayavki_bot
   ```
   Если такой уже занят — добавь цифру или придумай другой.
6. BotFather пришлёт сообщение с **токеном** — длинная строка вида:
   ```
   8197234567:AAH3xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
   **Это секрет.** Никому не показывай, в чат не отправляй. Скопируй в надёжное место — он понадобится дальше.

---

## Шаг 2. Получить chat_id целевого чата

Заявки могут приходить:
- **тебе в личку** — самый простой вариант, если ты один обрабатываешь заявки;
- **в групповой чат** — если несколько человек должны видеть заявки (ты + папа, например).

### Вариант А — в личку

1. В Telegram открой созданного бота (через ссылку из BotFather или через поиск по юзернейму).
2. Нажми **«Запустить»** (отправит `/start`).
3. В поиске набери **`@userinfobot`**, открой, нажми «Запустить».
4. Бот пришлёт твой `id` — целое число, например `123456789`. **Это и есть chat_id.**

### Вариант Б — групповой чат

1. Создай в Telegram новую группу (или открой существующую).
2. Добавь в неё своего бота (через поиск по юзернейму) и любого второго человека (Telegram не разрешает создавать группу из одного участника).
3. Внутри группы отправь любое сообщение (например, `/start`).
4. Открой в браузере (или мобильном Safari/Chrome):
   ```
   https://api.telegram.org/bot<ТВОЙ_ТОКЕН>/getUpdates
   ```
   Замени `<ТВОЙ_ТОКЕН>` на токен из Шага 1.
5. В ответе найди блок `"chat":{"id":-1001234567890,...}`. Число со знаком минус — это `chat_id` группы. **Запиши его.**

> Если в `getUpdates` пусто — отправь в группу ещё одно сообщение и обнови страницу.

## Шаг 3. Что делать дальше

После того как у тебя есть:
- ✅ **TG_BOT_TOKEN** (из Шага 1)
- ✅ **TG_CHAT_ID** (из Шага 2)

Переходи к [worker/README.md](worker/README.md) — там инструкция, как задеплоить обработчик формы в Cloudflare и подключить эти значения.

## Что если что-то пошло не так

**Бот не отвечает / нет ответа от `/start`** — возможно, ты не нажал «Запустить». Перейди в чат с ботом и нажми кнопку «Запустить» (или отправь `/start`).

**`getUpdates` возвращает `{"ok":true,"result":[]}`** — Telegram ещё не зарегистрировал сообщения для бота. Отправь в группу ещё одно сообщение и попробуй снова через минуту. Команда показывает только последние ~24 часа.

**Получаешь ошибку `Unauthorized`** — токен неверный или скопирован с лишним пробелом. Запроси токен у BotFather командой `/mybots` → выбери своего бота → «API Token».

**Хочу регенерировать токен** — `/mybots` в BotFather → выбери бота → «API Token» → «Revoke current token». Старый токен сразу перестанет работать, нужно будет обновить секрет в Cloudflare.

**Бот не может писать в группу** — открой настройки группы → «Управление группой» → «Администраторы» → дай боту права администратора с правом отправки сообщений.
